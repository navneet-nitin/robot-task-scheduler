Task Scheduling for the Robotic Assistant

Meet RoboX, your futuristic robotic assistant! RoboX performs a series of tasks, each requiring increasing amounts of time: 1 hour for the first, 2 hours for the second, 3 hours for the third, and so on. Given a total number of available hours, how many full tasks can RoboX complete?

Try it yourself:

Java Implementation

class RoboX {
    public int TheFuture(int n) {
        int start = 1, end = n;
        while (start <= end) {
            int mid = start + (end - start) / 2;
            long sum = (long) mid * (mid + 1) / 2;
            if (sum == n)
                return mid;
            else if (sum < n)
                start = mid + 1;
            else
                end = mid - 1;
        }
        return end;
    }
}

Time & Space Complexity

Time Complexity: O(log n) — due to binary search.

Space Complexity: O(1) — constant space used.
